Pi constant: Since PI is a constant, which is also a number, it is treated like any of the number keys, and can be used in non trigonometric calculations, i.e. calculations not involving angles.

Issues with PI in trigonomic functions
Since PI = 3.14 radians (to 2.d.p) and the trigonometric functions interpret the entered parameter as degrees rather than radians, using the PI button in a trigonometric function will give the wrong answer. Since, for example, keying in 'sin + PI', which displays as 'sin(3.14....)' is using the value of PI as a constant rather than a radian value; PI is treated as 3.14degrees rather than 3.14 radians (i.e. approximately 180 deg). So that, instead of returning the sine of PI as sine of 180deg (which is 0)

Example:

you get the sine of PI as sine of 3.14deg (approximately 0.06)

Example:
which, I assume, the user isn't looking for. So it is advisable for now to avoid using the PI button for trigonomic calculations if you wish to have 180degrees somewhere in your calculations. I'll think about changing the parameter for the trigomomic functions to radians in a future iteration of the calculator.

Natural Log function: The natural log function is simply carried out by pressing the 'LN' button and followed by the number for which you want to take the natural log of, and then pressing the '=' button

Prime number checker: To check whether a number is prime, first enter a number, and then press the 'Prime' button. If the number is prime, then in the smaller display the words 'Prime Number' will appear

Example:

If the number is not prime there are two possible outcomes; either the number is an even number that is not two, or it is not even, but has has a factor pair that does not include the number '1' or the number you entered (i.e. has more than one factor pair). If the number is even then the upper display show the words 'muliple of 2' followed by 'NP' (meaning 'not prime')
