Reading the display!

Main Chronograph readout: The main readout, which is the larger, bottom readout, displays the time elapsed since the start button was pressed or the time at which the pause button was pressed. When pause is pressed the main readout opacity changes so that the digits are somewhat faded and flash intermittently

Split readout: The split readout displays a stationary time, either zero time, which represents the state of the chronograph before the start button is clicked, or the last time at which the lap button was pressed. It essentially gives you the end-time of the last lap, so that, each time the lap button is pressed, this (split) readout updates to show precisely when that happened, displaying the last lap's end-time

Lap readout: On pressing the lap button, the lap readout is refreshed and displays a new time, the duration of the last lap (which completed when you pressed the button), and also the lap number, in the small rectangular box to the left of the lap time.

So, for instance, if the lap readout shows , that means you are currently on lap 4, and the last completed lap, lap 3, is now being displayed along with the time (approximately 58 seconds) that it took to complete that lap. Of course, there's another scenario, where you may have paused or reset the chronograph which means you are no longer timing laps and so the lap readout is actually displaying the final lap number and its duration (since no other laps will be timed, and hence the displayed lap 'was' the final lap).
