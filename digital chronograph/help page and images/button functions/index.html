What the Buttons do!

Start Button: The start button activates the stopwatch; the button colour will change to red and remain so while the stopwatch is running - if paused, the play button will revert to the original grey colour, while pause will change to red - to deactivate pause and start the stopwatch running again, just press the start button once more

Pause Button: The pause button halts the stopwatch. When pressed the large digits readout will stop counting, then flash on and off, and its opacity will reduce slightly making it look somewhat faded. To unpause the time, press the start button again - or if you wish you can reset the chronograph to zero-time while in pause mode.

Reset Button: The reset button resets the stopwatch to zero. In order to activate the reset button, you must first press the 'pause' button. This is done to prevent accidental activation of the reset function while the stopwatch is counting time.

Lap Button: The lap button comprises two operations in one that happen simultaneously. The first operation shows you the 'split time' , which is the exact moment you pressed the lap button, in other words, exact time the last lap was completed.

The second operation shows you the 'lap time' ,which is how long the lap actually took to complete, and also the lap number in the small rectangular box to the left of the lap time. Each time the lap button is pressed these two readouts are updated. Further to this update, the lap duration is saved behind the scenes (since the previous lap time disappears when a new lap time is displayed); previous laps can then be viewed later, by pressing the RECALL button. For a more in-depth description on how to read the display, open the Reading The Display page.

Lap RECALL: The RECALL button is initially greyed out and unavailable when the chronograph is first started. It only becomes available once the LAP BUTTON has been clicked once. The recall function cycles through all instances of lap updates. This can actually be done while the clock is still running, so the user can check previous lap times while still using the chronograph to record more laps; the previous laps can be viewed, and then when the lap button is pressed again, the record will be updated so that the new lap is included in the list of laps you can cycle through. The RECALL function is also available if the chronograph is paused, and even after the chronograph has been reset to zero time, but prior to the pressing of the LAP RESET button.

Lap RESET:The RESET button is initially greyed out and unavailable when the chronograph is first started. In order for the LAP RESET button to be useable, two conditions must exist. Firstly, saved laps must exist, and secondly, the chronograph must be RESET to zero. When these two conditions exist the lap reset button will flash with a light blue colour indicating to the user that the lap reset function is now available. At this point (prior to pressing the button) you can still use the lap recall button to view all recorded laps. Clicking the lap reset button will then erase all lap data and cause both the lap button and lap recall buttons to become slightly greyed out and unclickable. In this state, the chronograph is ready to record a completely new event.
