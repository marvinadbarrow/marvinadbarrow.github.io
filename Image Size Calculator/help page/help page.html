User Guide

The file chooser: To get images to work with, start by clicking the red choose file button

The select box: A select-file box will pop up so you can choose an image you want to get new dimensions for (you can only select one file at a time in the box - so repeat this process if you need to load more than one file). Double click an image or click an image and click the 'open' button.

The image shelf: After opening the image, it will appear in the 'image shelf' which is the dark grey area below the blue bar. You can select up to 'four' images as shown below - trying to select more than four will give an 'alert' popup on screen telling the user that the maximum amount of images selected has been reached. Refresh the page to select further images

four images selected

Pick an image: Clicking an image in the image shelf will highlith it with a thick dark blue border, and display a copy of it in the upper section above the blue bar, directly under the 'choose file' button. The image name and image size, with dimensions of Width X hEIGHT will appear inside the blue bar

Dimension choosers:The elements with the input fields of 'new width' and 'new height' can be used to determine new dimensions for the image you have selected.

input new width: Now, let's say you have an image with dimensions 1029 x 1008, but you need its width to be 1000; you don't know what height it should be to maintain the aspect ration. Type 1000 into the new width input and then click the 'get height' button, and the new dimensions of width x height (1000 x 979), representing the resized dimensions of the selected image will appear in the box below the 'get height' button; you can then use CSS to size the image for your webpage.

input new height:The same can be done if you wanted to get a new image size by dictating what the new height should be. Say you you need the image's height to be 500; you don't know what width dimensions the image should have to maintain the aspect ratio. Type 500 into the 'new height' input and then click the 'get width' button, and the new dimensions of width x height (510 x 500) will appear below that button; you can then use CSS to size the image for your webpage.

Both height and width:You don't need to limit yourself to just one dimensions. You can check both dimensions, width and height on the same image to get two different resizings, one based on a new height and the other based on a new width.
