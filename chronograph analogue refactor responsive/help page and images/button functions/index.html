What the Buttons do!

Play Button: The play button activates the stopwatch; the button colour will change to red and remain so while the stopwatch is running - if paused, the play button will revert to the original blue colour, while pause will change to red - to deactivate pause and start the stopwatch running again, just press the play button once more

Pause Button: The pause button halts the stopwatch. When pressed it will turn red and all stopwatch hands will stop. To unpause, press the play button again - or if you wish you can activate reset while in pause mode.

Lap Button: when the lap button is pressed, all stopwatch hands will stop, giving you the the exact time at which it was pressed. You can use it to measure the 'end time' of the particular lap you are on; you can then record the exact time - writing it down for example. The stopwatch continues to run in the background while the lap button colour is red; to start seeing the realtime again, just hit the lap button which will turn blue, and all hands will 'catch up' to the current elapsing time. The lap button only works while the play button is activated.

Refresh Button: The refresh button resets the stopwatch to zero. In order to activate the refresh button, you must first press the 'pause' button. This is done to prevent accidental activation of the reset function while the stopwatch is counting time.

Hide Clock Hands: Looking at the chronograph face it might have occured to you that the subdials are sometimes obscured by the main clock hands; this makes it difficult, or sometimes impossible, to read from the subdials. To resolve this problem I've added the 'hide clock hands' button. Activating it while the stopwatch is running or is in in lap mode will cause it to change colour to yellow, and the large hour and minute hands to start flashing and remain almost transparent, allowing you to see and read any of the dials obscured by those hands. The revolutions of those hands will not be affected and they will continue to keep normal clock time. Just hit the 'hide clock' button again if the large hands no longer obscure the subdials, and will revert to the default colour
